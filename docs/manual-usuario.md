# Manual de usuario

## Objetivo

Crear un plan mensual de alimentacion saludable para una o varias personas, con recetas, validacion nutricional y compra organizada.

## Pasos de uso

1. Indicar el numero de personas.
2. Elegir si el mes tiene 30 o 31 dias.
3. Introducir el presupuesto mensual maximo.
4. Marcar restricciones:
   - vegetariano;
   - vegano;
   - sin gluten.
5. Escribir alergias separadas por comas si existen.
6. Activar o desactivar preferencias culinarias.
7. Pulsar `Generar plan`.

## Que muestra la aplicacion

- Resumen economico del mes.
- Cobertura media de objetivos nutricionales.
- Graficos de energia, proteinas y micronutrientes.
- Tabla de validacion por nutriente.
- Calendario mensual por dias.
- Recetas detalladas del dia seleccionado.
- Disponibilidad del ingrediente en Mercadona y Consum de Paiporta.
- Lista de la compra agrupada por secciones.

## Interpretacion del informe

- `Cobertura %`
  - 100% o mas significa que el plan alcanza o supera el objetivo configurado.
- `Dentro del presupuesto`
  - el coste estimado queda holgadamente bajo el limite.
- `Muy ajustado`
  - el plan entra dentro del presupuesto pero con poco margen.
- `Por encima del presupuesto`
  - conviene reducir preferencias costosas o cambiar restricciones.

## Recomendaciones de salud

- El sistema sirve como apoyo de planificacion, no como diagnostico medico.
- Para embarazo, infancia, enfermedad renal, diabetes insulinodependiente o patologias digestivas complejas, validar con un dietista-nutricionista colegiado.
- Mantener hidratacion adecuada y actividad fisica regular.
